# УСТАНОВКА
###### Предисловие: для корректной установки необходим python3. Подойдет любая подверсия, так как виртуальное окружение в любом случае установит необходимый.
### Создание виртуального окружения
##### Команды
* Windows:
    `python -m venv venv`
    `source venv\scripts\activate.bat`
* MacOS/Linux:
    `python3 -m venv venv`
    `source venv/bin/activate`
##### Пояснения
*
    Данные команды создают и активируют виртуальное окружение. Оно, как я уже говорил раньше, является некой лабораторией, необходимой для беспрепятственного переноса проектов на другие компьютеры в пару консольных команд. Туда будут устанавливаться все библиотеки и таким образом всё, что нужно для корректной работы программы, будет в одной папке.

### Установка необходимых библиотек
##### Команда
`pip install -r requirements.txt`
##### Пояснение
*
    После написания всего кода, используя команду `pip freeze`, можно получить список всех используемых библиотек и их версий, а после - добавить их в файл requirements.txt, чтобы вышеуказанной командой установить все необходимые библиотеки одной командой.
    
### Запуск программы
*
    Файл main.py запускает работу программы для дальнейшего изучения её внешности и внутренностей.
